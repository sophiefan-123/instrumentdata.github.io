§ 01 — About the Project

Global String Instrument Properties

This project investigates the properties of string instruments across cultures and continents, using a comprehensive global database. By analyzing variables such as string count, tuning, scale length, and material, we aim to identify patterns that reveal how different regions develop similar or distinct instrument characteristics.

Comparing instruments from Asia, Africa, the Americas, Europe, and Oceania allows us to explore the connections between musical design and culture, including ecological conditions, trade, and tradition.

Findings highlight regional similarities and differences, encouraging reflection on how cultural practices shape the instruments that define musical expression worldwide. Through this lens, we ask: what do instruments reveal about the societies that built them? How do trade routes, ecological constraints, and ritual practice shape the physics of sound?
