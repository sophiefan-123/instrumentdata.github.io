Variable Codebook

Key variables recorded for each instrument entry, their data types, and example values.

Variable Type Description Example Values
instrument_name Text Primary name in source language with transliteration Sitar, Kora, Oud
hs_classification Categorical Hornbostel-Sachs classification code 321.322, 312.12
region Categorical Broad geographic region of cultural origin South Asia, West Africa, Middle East
country Text Country of primary association India, Senegal, Morocco
string_count Numeric Total number of strings including drone strings 4, 7, 21
tuning_system Categorical Primary intonation system used Just intonation, Pentatonic, Equal temperament
resonator_material Categorical Primary material of the resonating body Gourd, Wood, Metal
ensemble_context Categorical Primary performance context Solo, Ensemble, Ceremonial
